Um pouco sobre o site

Esse site foi criado com HTML, CSS e JS apenas, nenhuma ferramenta foi inserida na otimização do codigo, particularmente preferi assim, pra que eu pudesse cada vez mais entender as mecanicas de criação de site sem atalhos e conter o maximo de evolução durante o aprendizado.

Por que criei este site?

Este site foi criado para me ajudar a otimizar o meu tempo, sempre tinha dificuldades para lembrar de vídeos ou assuntos que estava acompanhando diariamente, com a criação desse site, pude colocar links para vídeos, que muita das vezes, não conseguia concluir de uma vez só, então tive a idéia de colocar esses links para que não me perdesse mais nos estudos, sendo assim, otimizando o meu tempo apenas com um click!

Consegui otimizar o meu tempo

Com certeza!?, a partir do momento em que criei este site, a cadência dos estudos ficaram cada vez melhor, porque posso atualizar os links a cada semana e passar a semana entrando e assistindo com um click a cada dia, obviamente requer um tempo para atualizar os links, mas o tempo gasto para isso, é extremamente menor do que procura-los no YouTube.
